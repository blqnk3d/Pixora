# Prity - Pixel Art Editor

A lightweight, browser-based pixel art editor built with vanilla JavaScript and Canvas API.

## Features

- Drawing tools: Pencil, Eraser, Fill, Selector
- Layer system with opacity and visibility controls
- Color picker with palette and recent colors
- File operations: New, Open (PNG/JPG/GIF), Save PNG
- Undo/redo support
- Background removal tool
- Zoom, grid toggle, dark/light theme
- Keyboard shortcuts

## Usage

Open `index.html` in a browser or host on GitHub Pages.

## Keyboard Shortcuts

- `B` - Pencil tool
- `E` - Eraser tool
- `G` - Fill tool
- `M` - Selector tool
- `Ctrl+S` - Save PNG
- `Ctrl+O` - Open file
- `Ctrl+Z` - Undo
- `Ctrl+Y` - Redo
- `Ctrl+Scroll` - Zoom
